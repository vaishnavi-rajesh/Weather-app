This is a simple console-based Weather Application that allows users to enter a city name and view basic weather details such as temperature and weather condition.

## Features
- Fetches real-time weather data using OpenWeatherMap API
- Displays temperature and weather condition
- Simple and clean console interface

## Technologies Used
- Python
- OpenWeatherMap API
- Requests library

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python weather.py
